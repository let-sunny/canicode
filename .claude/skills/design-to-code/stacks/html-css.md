# Stack: HTML + CSS

Single standalone HTML file, no build step.

## Conventions
- Semantic HTML elements (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`)
- CSS variables for colors
- Flexbox / Grid for layout
- All styles in `<style>` tag (no external CSS file)

## Output filename
`index.html`

## Template
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <link href="https://fonts.googleapis.com/css2?family=FONT_NAME&display=swap" rel="stylesheet">
  <style>
    /* styles here */
  </style>
</head>
<body>
  <!-- design tree → HTML elements -->
</body>
</html>
```
